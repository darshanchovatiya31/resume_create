## How to Regenerate PDF

### Version 1:
1. Make your edits to `resume.html`
2. Run:
   ```bash
   node generate-pdf.js
   ```

### Version 2:
1. Make your edits to `resume-v2.html`
2. Run:
   ```bash
   node generate-pdf-v2.js
   ```

### Version 3:
1. Make your edits to `resume-v3.html`
2. Run:
   ```bash
   node generate-pdf-v3.js
   ```

## Alternative: Browser Print to PDF

You can also generate a PDF directly from the HTML file:

1. Open `resume.html` in your web browser
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Select "Save as PDF" as the destination
4. Click "Save"
